# Odoo 18 Custom Modules

這個資料夾整理了 2026-04-22 針對 Odoo 18 的兩個自訂模組。

## 模組列表

### `sale_sales_slip`

銷售單相關自訂模組，提供銷貨單列印與欄位調整。

### `tw_untaxed_label_fix`

台灣發票與報表調整模組，包含：

- 將 `未連稅金額` 改為 `未稅金額`
- 移除發票與報價單中的不需要 footer / Tax ID / 付款通訊內容
- 新增台灣版 `電子發票證明聯`
- 調整發票報表版面接近台灣使用習慣
- 新增發票表單欄位 `報表備註`
- 電子發票 PDF 備註區只顯示 `報表備註`，不再自動帶入內部單號

## 部署位置

Odoo 主機自訂模組路徑：

`/opt/odoo/custom_addons`

## 更新方式

更新模組後，需在 Odoo 主機執行模組升級並重新啟動 Odoo 服務。
